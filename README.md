# Backend

## Install

```shell
npm i
```

Create ***.env*** file using ***.env.example*** as template

## Run

development mode

```shell
npm run dev
```

production mode

```shell
npm start
```

## Testing

Run tests

```shell
npm test
```

Watch testing coverage

```shell
npm run cover
```
